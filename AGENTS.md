# BioRevive Lab - Agent Knowledge Base

## Core Product: จุลินทรีย์เชื้อสด (Microbial Solution)
- **Technology**: Active 99% Microbial Solution (BCG Model).
- **Key Enzymes**:
  - **Lipase (ไลเปส)**: ย่อยสลายไขมัน (Fats/Oils/Grease).
  - **Protease (โปรตีเอส)**: ย่อยโปรตีน (Protein).
  - **Amylase (อะไมเลส)**: ย่อยแป้ง (Starch).
- **Benefits**: Odor control within 24h, prevents drainage blockages, eco-friendly, cost reduction.

## Usage & Application
- **Pipes/Sinks**: 50g per week. Mix with water and pour into sinks/drains.
- **Grease Traps**: 1kg per 10 cubic meters per week. 
- **Precautions**: 
  - Remove hard grease crusts before first application.
  - Do not use with strong chemicals, bleach, or high amounts of dish soap (kills microbes).
  - Apply consistently every week.
