# BioRevive Lab Project Guide

This file contains specific rules for the BioRevive Lab website project.

## Branding & Business Names (CRITICAL)

- **Business/Shop Name (For Documents)**: **BioRevive Thailand - ไบโอรีไวฟ์ ไทยแลนด์** (Use this for Quotations, Invoices, Letterheads).
- **Brand/Marketing Name**: **BioRevive Lab - ไบโอรีไวฟ์ แลปส์** (Use this for Website, Bot, and Product labels).
- **Forbidden Name**: Never use "BioRevive Pro".
- **Keywords**: BCG Model, ESG, Active 99% Live Microbes, Bio-Circular-Green Economy.

## Product Taxonomy
1. **จุลินทรีย์เชื้อสด สูตรพรีเมี่ยม (Active Microbes - Premium)**
   - Sizes: 1 kg. / 6 kg. / Sachets (20 g. x 6)
   - Usage: Heavy problems, initial reset, continuous care.
2. **จุลินทรีย์เชื้อสด สูตรน้ำ (Active Microbes - Liquid)**
   - Sizes: 1 L. / 5 L.
   - Usage: Daily maintenance, odor control, mopping.

## Tone and Voice
- Professional, premium, yet accessible.
- Emphasis on "Dream-Risk-Solution" storytelling.
- Focus on sustainability (BCG/ESG).

## Technical Context
- Primary platform: HTML/CSS/JS (Static) with Python (Flask) for LINE Bot API.
- Deployment: Vercel.
- Integration: LINE Official Account (@biorevivelab).
